# 🧪 Sistema de Reserva de Laboratório 

O projeto consiste em um sistema feito para professores e monitores solicitarem horários de uso dos laboratórios do DCET. O objetivo é facilitar o agendamento, dar visibilidade pública à agenda dos laboratórios e reduzir o trabalho manual do administrador responsável (hoje feito por e-mail e planilha).
